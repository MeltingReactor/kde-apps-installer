# KDE Apps Installer
A bash script for arch linux to install default kde apps found on Fedora KDE.

> [!NOTE]
> This script will only work on **Arch Linux**.

---
[Download](https://github.com/MeltingReactor/kde-apps-installer/releases/latest/download/KInstall.sh) and run `KInstall.sh` to install all of the default KDE Plasma apps.
